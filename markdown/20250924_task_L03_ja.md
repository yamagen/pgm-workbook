# Lesson 3: 調整文法とは何か（構造と時間）

## 1. 調整文法の考え方

- 調整文法は、時間をかけて**文の構造（シンタクス）**を組み立てる文法である。
- 即時文法が「順番」で成り立つのに対して、調整文法は「構造」で成り立つ。
- 書き言葉、説明、発表などで特に使われる。

---

## 2. 調整文法の特徴

- 主語・述語・修飾語などの関係を意識する。
- 助詞（が・を・に・で など）を正しく使う。
- 文を「最後まで完成」させる。

**例**

- 「私はきのうお店に行きました」
- 「このケーキはとてもおいしかった」
- 「雨が降っていたので、外出しませんでした」

---

## 3. 即時文法との対比

| 特徴   | 即時文法                           | 調整文法                       |
| ------ | ---------------------------------- | ------------------------------ |
| ルール | 発話の順番                         | 統語構造（シンタクス）         |
| 時間   | 短い、すぐに出る                   | 長い、準備して出す             |
| 文末   | 「よ」「ね」などで終わることもある | 述語で文を完結させる           |
| 完全性 | 不完全でもOK                       | 完全な文を目指す               |
| 例     | 「おみせ、いった、きのう」         | 「私はきのうお店に行きました」 |

---

## 4. 使い分け

- **即時文法** → 会話の流れ、感情表現、あいづち
- **調整文法** → 説明、発表、論文、レポート

**ポイント:**  
日本語は、この二つが状況に応じて切り替わる。
どちらも大切で、使い分けが自然な日本語を作る。

## 6. まとめ

- 調整文法は「時間をかけて構造を組み立てる文法」である。
- 助詞や述語を使い、文を最後まで完成させる。
- 即時文法と調整文法の両方を知ることで、会話も文章もバランスよく使える。
